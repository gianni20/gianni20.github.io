Library Management


Sistema di gestione biblioteca con React e Django

Il Progetto

Questo progetto consiste in un sistema completo e intuitivo per la gestione operativa di una biblioteca. Include flussi per la registrazione di prestiti e restituzioni di libri, la gestione di anagrafiche utenti e il monitoraggio in tempo reale della disponibilità dei volumi a catalogo.

L'applicazione è stata concepita con un'architettura disaccoppiata (Decoupled Architecture), dove un frontend reattivo interagisce con un server backend RESTful centralizzato tramite chiamate API asincrone sicure.

Tecnologie Utilizzate

  • React.js
  • Django REST Framework
  • PostgreSQL Database
  • Bootstrap CSS
  • RESTful API Calls

Caratteristiche Principali

  • Gestione e tracciamento prestiti e restituzioni
  • Sistema di autenticazione e autorizzazione utenti
  • Pannello amministrativo (dashboard) di monitoraggio
  • Filtri di ricerca avanzati per titoli, autori e generi
  • Notifiche automatiche per ritardi nelle consegne
  • Profilo personale utente con storico attività
