Library Management

Sistema di gestione biblioteca con React e Django

Il Progetto

Un sistema completo per la gestione di una biblioteca, che include funzionalità per il prestito di libri, la gestione degli utenti e il monitoraggio delle disponibilità.

Il progetto è stato creato al fine di creare un'interfaccia in grado di rispondere e interagire con un database esterno tramite API calls.

Tecnologie Utilizzate

  • React
  • Django
  • PostgreSQL
  • REST API
  • Bootstrap

Caratteristiche Principali

  • Gestione prestiti e restituzioni
  • Sistema di autenticazione utenti
  • Dashboard amministrativa
  • Ricerca avanzata libri
  • Notifiche ritardi prestiti
  • Profilo personale
